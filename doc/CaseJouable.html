    class CaseJouable

    La classe CaseJouable :

    • représente une case sur laquelle on peut cliquer

    Les VI de la classe sont :

    • @etat ==> indique si la case est une case ile, océan ou non jouée

      0    = case non jouée
1    = océan
2    = île
